AuthentiScan - Image Forensic Analysis Tool

★★★★★

AuthentiScan is an AI-powered image authenticity verification system developed by John Chris Idioma and Kenn Carrido. The system analyzes uploaded images using Error Level Analysis (ELA), EXIF metadata inspection, compression ratio comparison, and a Convolutional Neural Network (CNN) to determine whether an image is authentic, manipulated, or AI-generated. It also estimates the image's likely source platform and generates a comprehensive forensic report, helping users verify digital image authenticity and combat the spread of manipulated content.
